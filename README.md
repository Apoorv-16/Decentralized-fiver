# Decentralized Fiver

Decentralized Fiver is a Full Stack Data Labelling App.
Uses Phantom wallet to create data labelling tasks open to everyont to take it up and get paid.
## Tech Stack

**Client:** Next JS, TypeScript

**Server:** Node JS, Express JS

**Database:** Postgres
